## 1. Quyết định thông minh với If-Else
Trong lập trình mạng, một thuật toán phải biết đưa ra quyết định dựa trên dữ liệu thực tế. Cấu trúc `if-else` chính là "ngã ba đường" giúp chương trình định hướng.

```java
if (diemSo >= 5) {
    System.out.println("Chúc mừng, bạn đã vượt qua thử thách!");
} else {
    System.out.println("Kiên trì hơn một chút, thành công đang đợi phía trước.");
}
