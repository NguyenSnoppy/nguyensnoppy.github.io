Bai 7 Lap Trinh Mang Voi Java

Nội dung chính

Đây là phần cốt lõi của học phần. Java hỗ trợ lập trình mạng rất tốt thông qua gói java.net.

  • IP Address: Định danh của máy tính trong mạng.
  • Port: Cổng dịch vụ (ví dụ: 80 cho web, 443 cho https).
  • TCP/IP: Giao thức truyền tin tin cậy.

Kiến thức này giúp mình hiểu cách các ứng dụng như Facebook hay Zalo truyền nhận tin nhắn.
