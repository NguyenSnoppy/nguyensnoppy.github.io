Bai 1 Tong Quan Java

1. Java là gì?

Java là một ngôn ngữ lập trình hướng đối tượng (OOP) rất mạnh mẽ. Mình chọn Java vì nó có tính bảo mật cao và được dùng nhiều trong lập trình mạng.

2. Tại sao nên học Java?

  • Viết một lần, chạy mọi nơi (Write Once, Run Anywhere).
  • Cộng đồng hỗ trợ rất lớn.
  • Phù hợp để làm các ứng dụng Server-Side.

3. Cảm nhận cá nhân

Lúc đầu học mình thấy hơi khó vì cú pháp khắt khe, nhưng càng học càng thấy nó logic.
