Bai 1 Tong Quan Java

1. Java là gì? Java là một ngôn ngữ lập trình hướng đối tượng (OOP) rất mạnh mẽ. Mình chọn Java vì nó có tính bảo mật cao và được dùng nhiều trong lập trình mạng. 2. Tại sao nên học Java? Viết một lần, chạy mọi nơi (Write Once, Run Anywhere). Cộng đồng hỗ trợ rất lớn. Phù hợp để làm các ứng dụng Server-Side. 3. Cảm nhận cá nhân Lúc đầu học mình thấy hơi khó vì cú pháp khắt khe, nhưng càng học càng thấy nó logic.

Bai 2 Bien Trong Java

Nội dung chính Trong Java, việc hiểu rõ kiểu dữ liệu là rất quan trọng vì đây là ngôn ngữ “strong typed”. Kiểu số nguyên: int, long dùng cho các số không có phần thập phân. Kiểu số thực: double, float dùng cho các phép tính số dư, điểm số. Kiểu chuỗi: String dùng để lưu trữ văn bản. Kiểu logic: boolean chỉ nhận giá trị true hoặc false. Kinh nghiệm: Mình thường dùng int cho các biến đếm và String để quản lý thông tin người dùng trong các bài tập trên lớp.

Bai 3 Vong Lap Java

Nội dung chính Cấu trúc điều khiển giúp chương trình có thể đưa ra quyết định. If-else: Kiểm tra điều kiện (ví dụ: nếu điểm >= 5 thì đậu). Vòng lặp For: Dùng khi biết trước số lần lặp (ví dụ: in danh sách sinh viên). Vòng lặp While: Dùng khi lặp dựa trên một điều kiện còn đúng. Ví dụ: Mình đã áp dụng vòng lặp for để duyệt qua danh sách các địa chỉ IP trong bài thực hành lập trình mạng.

Bai 4 Javascript Co Ban

Nội dung chính Nhiều người lầm tưởng Java và JavaScript giống nhau, nhưng thực tế chúng rất khác biệt. Java: Cần biên dịch, chạy trên máy ảo JVM, phù hợp cho hệ thống lớn. JavaScript: Ngôn ngữ kịch bản, chạy trực tiếp trên trình duyệt, giúp web tương tác tốt hơn. Ứng dụng: Trong đồ án này, mình dùng JS để tạo các hiệu ứng mượt mà cho giao diện Blog tối giản.

Bai 5 Dom Trong Js

Nội dung chính DOM (Document Object Model) là cầu nối giữa JS và HTML. Cách lấy phần tử: document.querySelector(). Lắng nghe sự kiện: addEventListener('click', ...). Thông qua DOM, mình có thể tạo ra các nút bấm đổi màu nền hoặc hiện thông báo khi người dùng tương tác với Blog.

Bai 6 Async Await Js

Nội dung chính Trong lập trình mạng, dữ liệu không phải lúc nào cũng có ngay lập tức. JavaScript giải quyết vấn đề này bằng: Promise: Lời hứa về một giá trị trong tương lai. Async/Await: Cú pháp giúp code bất đồng bộ trông giống như code tuần tự, dễ đọc hơn. Việc nắm vững kiến thức này giúp mình xử lý dữ liệu từ API hiệu quả mà không làm treo trình duyệt.

Bai 7 Lap Trinh Mang Voi Java

Nội dung chính Đây là phần cốt lõi của học phần. Java hỗ trợ lập trình mạng rất tốt thông qua gói java.net. IP Address: Định danh của máy tính trong mạng. Port: Cổng dịch vụ (ví dụ: 80 cho web, 443 cho https). TCP/IP: Giao thức truyền tin tin cậy. Kiến thức này giúp mình hiểu cách các ứng dụng như Facebook hay Zalo truyền nhận tin nhắn.

Bai 8 Ung Dung Chat Don Gian

Nội dung chính Socket là một đầu cuối (endpoint) trong mối liên kết giao tiếp giữa hai chương trình chạy trên mạng. ServerSocket: Chờ đợi kết nối từ phía khách. Socket: Dùng để gửi và nhận luồng dữ liệu (InputStream/OutputStream). Trong bài thực hành, mình đã thử tạo một chương trình Server đơn giản có thể phản hồi lại tin nhắn từ Client.
